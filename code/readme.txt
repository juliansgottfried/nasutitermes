Analyze.java runs Sim.java over as many iterations as specified, collecting and returning data.

Sim.java can be run as many times as desired, and can be animated.

Build.java collects user input to make a map on which Sim.java can be run.

SpineTree.java, Spine.java, Termites.java, ResizingArray.java, and BuildRandom.java are all helper classes for the above.
